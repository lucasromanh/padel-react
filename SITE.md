# Site Vision: Padel Club Management System

A premium, responsive web system for a single padel club. The system manages court reservations, matches, tournaments, and live scoreboards.

**Project ID:** 18165195955492391362

## 1. Site Vision
Provide a seamless experience for players to book courts and track their progress, while giving club operators powerful tools to manage the daily business and tournaments.

## 2. Navigation Structure
- `/index.html` (Home)
- `/reservas.html` (Reservations)
- `/partidos.html` (Matches)
- `/torneos.html` (Tournaments)
- `/tablero-general.html` (Live Scoreboard - General)
- `/tablero-cancha.html` (Live Scoreboard - Court)
- `/ranking.html` (Club Ranking)
- `/login.html` (Login / Register)
